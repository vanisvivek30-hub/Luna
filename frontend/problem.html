Carbon Footprint Tracker

Easy ✦ New Full-Stack

About This Problem

Build a web application that helps small businesses track, visualize, and reduce their carbon footprint across energy usage, transportation, and waste categories. The app should provide actionable insights and monthly reports.

Requirements

  • User authentication and multi-company support
  • Dashboard with real-time emissions visualization (charts)
  • Category-based input forms (energy, travel, waste)
  • Monthly PDF report generation
  • RESTful API backend with a proper database schema

Tech Stack Suggestion

React / Vue frontend, Node.js or Django backend, PostgreSQL, Chart.js or D3 for visualizations.
